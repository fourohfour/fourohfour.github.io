Welcome to the homepage of FourOhFour

I'm FourOhFour, a programmer and computer science enthusiast from the United Kingdom. My username is derived from the infamous HTTP response code 404 - Not Found.

Current Projects

I have recently been working on readit - the result of me dabbling in the python web server framework flask. It is designed to provide an overview of the top submissions to reddit.
I am also working on Cayley - named after aviation pioneer Sir George Cayley - an aerodynamically accurate flight simulator within minecraft, running as a Bukkit Plugin. Repository to come in the future.

Previous Work

I have developed several bukkit plugins, including the Helix PvP minigame for the now defunct AkaneMC server network. I have also developed several reddit bots, most notably DogeHelpBot which provided information to members of the Dogecoin virtual currency community.
